# assets 폴더 안내

이 상세페이지는 **아래 파일명으로 이미지가 존재하면 자동으로 사용**합니다.
파일이 없으면 임시 플레이스홀더가 표시됩니다.

## 필요한 이미지 파일명

- `hero.jpg` : 상단 히어로 배경(금속 텍스처/배경 이미지)
- `pouch-front.jpg` : 제품 전면 사진
- `pouch-set.jpg` : 제품 구성/세트 사진(2열 중 왼쪽)
- `pouch-back.jpg` : 제품 후면 사진(2열 중 오른쪽)
- `pouch-open.jpg` : 내부 오픈 사진
- `detail-1.jpg` : 디테일 1
- `detail-2.jpg` : 디테일 2
- `detail-3.jpg` : 디테일 3
- `detail-4.jpg` : 디테일 4
- `keytag.jpg` : 키택/QR 등 세로형(아래 큰 사진)

## 팁

- 원본 페이지와 동일하게 보이게 하려면, **가급적 가로 1200px 이상(또는 원본 고해상도)** 이미지를 권장합니다.
- 이미지는 자동으로 `object-fit: cover`로 크롭되어 보이므로, 중요한 피사체는 중앙에 오도록 편집하면 좋습니다.
